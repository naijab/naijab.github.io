# naijab.com's blog

> **Just a lonely man likes coding** build with nuxt.js

## Build Setup

- `yarn` install dependencies

- `yarn dev` serve with hot reload at http://localhost:3000

- `yarn build && yarn start`  build for production and launch server

- `yarn generate` generate static project

- `yarn deploy` deploy to github pages

## Lint

- `yarn lint` for running linter
