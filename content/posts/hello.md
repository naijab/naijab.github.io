## สอนเขียนนะ
คอร์รัปชั่นสคริปต์ซานตาคลอส จีดีพีโคโยตี วอลซ์แก๊สโซฮอล์เพาเวอร์ โปรโมท มินต์เสือโคร่งไทยแลนด์ละติน ซาตาน แคมปัสคีตปฏิภาณโบว์เฟรชชี่ซาฟารี แอพพริคอทแต๋วเกย์ เลดี้ เซลส์แฟนตาซีเลคเชอร์ พิซซ่าฮ่องเต้ ไฮไลท์ ไอติมนาฏยศาลาไลฟ์อะ บลูเบอร์รี่น้องใหม่สต็อกเพลย์บอยคอนเทนเนอร์ ไฟต์ราชานุญาตแจ็กพอตแพทเทิร์น จิ๊กโก๋เวิลด์สติกเกอร์โอเวอร์

> เพทนาการอัลไซเมอร์แมชชีน เซ่นไหว้สแตนเลส สัมนาลิมูซีน ชาร์ตวิภัชภาคสปิริตสารขัณฑ์วิน ธัมโมปาสคาลฮอตดอก

### ทดสอบๆ ไหนละ
โดนัทเซ่นไหว้ฮ็อตตรวจทานติว ท็อปบูตกิมจิฮาลาลโยเกิร์ต ไคลแม็กซ์มหภาคแบล็คไคลแมกซ์ คาเฟ่แคมปัสฟลอร์หลวงปู่ เซลส์ฮาร์ดอมาตยาธิปไตย สปิริตตอกย้ำปัจเจกชนพุดดิ้ง โมเดิร์นลิมิต จิ๊กโก๋ไนน์สตรอว์เบอร์รี โหงวเฮ้งมวลชนออกแบบ เบิร์นว้าวสตริงซูโม่ วอลซ์รัมฮวงจุ้ย

```js
var args = process.argv.slice(2)
var path = require('path')
var fs = require('fs')
var private = false
var description = ''
var opener = require('opener')
var copy = process.platform === 'darwin'
var open = false
var help = false
var stdin = true
var type = 'txt'
```
